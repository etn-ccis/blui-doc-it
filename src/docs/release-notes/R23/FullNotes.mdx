### Documentation Site

-   Added design guidelines for [the Tables design pattern](/patterns/tables).
-   Added a color inspector to [the color palette page](/style/color). Now you may click on a color patch and view the color code in other color spaces and its color contrast ratios against other BLUI colors.

### Angular

-   Updated packages to support Angular / Angular Material v13.
-   Added [Toolbar Menu](https://brightlayer-ui-components.github.io/angular/components/toolbar-menu) component to existing component library.

### Ionic

-   Dropped support for Ionic.

### React

-   Updated packages to support MUI v5.
-   Updated packages to support React v18.
-   Added [Toolbar Menu](https://brightlayer-ui-components.github.io/react/components/toolbar-menu) component to existing component library.
