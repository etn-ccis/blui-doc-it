# Community Packages

The BLUI Community packages are auxiliary packages to the core Brightlayer UI platform that have been approved by the Brightlayer UI team. BL Charts Components and React Data Visualizations are two primary packages available.

## @blui-community/bl-charts-components

Customizable chart components built with Recharts and Material-UI (MUI), designed for flexible, responsive design and easy integration into new or existing applications.

## react-data-visualizations

This package includes React components for visualizations like Gauge, Stacked Gauge, and Dynamic Circle.

For detailed information on using each package, please refer to the [BLUI Community Packages Documentation site.](https://brightlayer-ui-community.github.io/)
