# Eaton Power BI Toolkit

## Introduction

The Eaton Power BI Toolkit is a subset of the [existing Power BI guidelines by Microsoft](https://learn.microsoft.com/en-us/power-bi/) and the [Brightlayer UI design system](https://brightlayer-ui.github.io/). The main goal of using Power BI reports in Eaton is to provide top priority insights in a clear and user-friendly manner, while leveraging and correlating data from different sources.

The Center for Intelligent Power team is committed to creating products that are user-centric, and we strongly recommend conducting user research to guide your design decisions in addition to using this toolkit.

## Why Should You Use the Eaton Power BI Toolkit?

The Eaton Power BI Toolkit is a collection of resources for designing Power BI reports. It includes design guidelines, a Power BI theme and layout templates.

Adhering to these guidelines and using our templates ensures:

- Consistency with other Eaton Power BI reports;
- Alignment to Eaton Branding standards;
- Faster cycle times for designing and publishing dashboards;
- An improved user experience for your audience.

For more detailed information, visit the [Getting Started Guide](https://eaton.sharepoint.com/sites/EatonPowerBIToolkit/SitePages/Home.aspx#).
